# HubSpot CRM Automation Testing Framework

This repository contains an automated test framework for validating the **HubSpot CRM product page** using:

- Java
- Selenium WebDriver
- Cucumber BDD
- TestNG & JUnit
- Maven
- Page Object Model (POM)
- Jenkins
